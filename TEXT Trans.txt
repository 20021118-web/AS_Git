3) 서비스 등록 — 한 줄씩 붙여넣기
기존 것 제거 (없으면 "Can't open service!" 떠도 무시):

C:\ServerBox\install\nssm.exe remove ServerBox-Gateway confirm
등록:

C:\ServerBox\install\nssm.exe install ServerBox-Gateway C:\ServerBox\gateway\caddy.exe run
작업 폴더 지정:

C:\ServerBox\install\nssm.exe set ServerBox-Gateway AppDirectory C:\ServerBox\gateway
자동 시작:

C:\ServerBox\install\nssm.exe set ServerBox-Gateway Start SERVICE_AUTO_START
시작:

C:\ServerBox\install\nssm.exe start ServerBox-Gateway
4) 확인
Start-Sleep 2; Get-Service ServerBox-Gateway; (Invoke-WebRequest http://localhost:8080/web/ -UseBasicParsing -TimeoutSec 5).StatusCode; "공유주소: http://" + (Get-NetIPConfiguration | Where-Object {$_.IPv4DefaultGateway -ne $null -and $_.NetAdapter.Status -eq 'Up'}).IPv4Address.IPAddress + ":8080/web/"
→ Status: Running + 200 + 공유주소: http://…:8080/web/ 나오면 완료입니다.

③에서 또 "액세스 거부"가 뜨면 창이 관리자가 아닌 것이니, 2)의 확인 명령이 True인지 먼저 봐주세요. 결과 알려주시면 다음 단계(팀원 공유 안내) 정리해 드리겠습니다.
