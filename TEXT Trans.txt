0) 부품 앱 파일 확인 (True 나와야 함)
Test-Path C:\ServerBox\apps\parts\main.py
False면 파일 복사가 덜 된 것 — 알려주세요.

1) 파이썬 있는지 확인
python --version
Python 3.11.x(또는 3.10+) 나오면 → 3번으로 건너뛰기
"인식되지 않습니다" 나오면 → 2번 진행
2) 파이썬 설치 (없을 때만)
winget install -e --id Python.Python.3.11 --accept-source-agreements --accept-package-agreements
설치 끝나면 이 관리자 창을 닫고, 관리자 PowerShell을 새로 여세요 (PATH 갱신). 그다음:

python --version
Python 3.11.x 확인.

winget 자체가 "인식 안 됨"이면 알려주세요 — python.org 설치 파일 방법을 드리겠습니다.

3) 가상환경 만들기
python -m venv C:\ServerBox\venvs\parts
4) pip 최신화
C:\ServerBox\venvs\parts\Scripts\python.exe -m pip install --upgrade pip
5) 패키지 설치 (수 분 소요 — 인터넷 다운로드 많음, 기다리세요)
C:\ServerBox\venvs\parts\Scripts\python.exe -m pip install -r C:\ServerBox\apps\parts\requirements.txt
→ 마지막에 Successfully installed ... 가 나오면 성공. (빨간 오류로 끝나면 그 내용을 알려주세요)

6) 서버 엔진 직접 실행해서 테스트 (한 줄)
C:\ServerBox\venvs\parts\Scripts\python.exe -m uvicorn main:app --app-dir C:\ServerBox\apps\parts --host 127.0.0.1 --port 8001
→ Uvicorn running on http://127.0.0.1:8001 가 뜨고 멈춘 상태가 되면 정상입니다.
그 상태로 서버 PC 브라우저에서 열기:

http://localhost:8001/
부품 매칭 화면(서버판) 이 뜨면 성공 → 터미널에서 Ctrl+C 로 멈추고 알려주세요.
